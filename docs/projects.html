Side Projects

  • Full-Stack e-Commerce Web Application: Developed a simple e-Commerce online shopping website as a full-stack web application by exposing inventory data from MySQL DB through REST APIs using a SpringBoot backend server which gets consumed by a frontend Angular client. Implemented features to display product data, search for products, add to shopping cart, checkout cart and form validation. In addition, security features were built by leveraging Okta with OAuth2.0.
  • Simple Task Tracker Web Application: This project centered around building a simple full-stack task tracker application. This is equipped with a simple user interface that allows the user to perform CRUD tasks.

Academic Projects at University at Buffalo

  • PintOS: Pintos is computer software, a simple instructional operating system framework for the x86 instruction set architecture. It supports kernel threads, loading and running user programs, and a file system, but it implements all of these in a very simple way. It was created at Stanford University by Ben Pfaff in 2004. I had Implemented synchronization primitives in C to enable kernel-level multithreading with priority scheduling. Developed 13 system calls in the PintOS kernel for an enhanced user-system interaction.
  • Simple Dynamo DB: Using Java on top of Android, I built a simple version of Dynamo DB, a distributed and replicated key-value storage using consistent hashing, with data replication, failure recovery and data synchronization.
  • Simple Distributed Hash Table: Using Java on top of Android, I implemented a simple version of a distributed hash table with consistent hashing.
  • Exploratory Data Analysis and Visualization with CDC flu data: This project aimed at collecting and visualizing flu data for the United States from CDC API. Data was collected across various times in massive sizes. Different charts explaining the trends of data were generated.
  • Data Aggregation, Big Data Analysis and Data Visualization: I had performed Map Reduce jobs for traditional Word Count and Word Co-Occurrence problems on Twitter, NY Times and Common Crawl data on Cloudera docker platform. Ranked the top 10 words of the results of Map Reduce for Word Count and Word Co-Occurrence on Tableau.
  • Measuring the degree of Robustness of Modern Convolutional Neural Networks : The project was aimed at the measurement of the degree of defensiveness of a convolutional neural network to adversarial examples (images). The idea is to generate slightly distorted images (attacks to a neural network) and measure the probability of its misclassification.
